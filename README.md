# General Purpose Front End Template

## Author: Davee Sok

## Links

- [Deployed link](https://practice1-nextjs.vercel.app/)

## Overview / Motivation

A general purpose front end project template meant to get up and running fast. Built with Nextjs, Material Ui, and Redux Toolkit. Comes with basic landing page, about page, signup, signin. Shows example of axios call to placeholder api and initializing redux store with data. Includes examples of getStaticProps, getStaticPaths, static site generated page, and client side rendered page.
